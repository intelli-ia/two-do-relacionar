# Método RELACIONAR — Landing Page

Landing page institucional para o projeto **Relacionar**, uma ferramenta de diagnóstico estratégico para empresas que buscam elevar sua maturidade em relacionamento, experiência e percepção de marca.

## 🚀 Sobre o Projeto

Este projeto foi construído utilizando **Next.js** e **Tailwind CSS**, seguindo a identidade visual premium da **TwoDo Design**. Ele apresenta o método RELACIONAR, seus 10 pilares fundamentais, a estrutura do diagnóstico e os níveis de maturidade empresarial.

## 🛠️ Tecnologias

- **Framework:** Next.js 15
- **Estilização:** Tailwind CSS (Vanilla CSS para design system core)
- **Componentes:** React (com micro-animações e Framer Motion/Custom IntersectionObservers)
- **Tipografia:** Mandalika (Display), Afterglow (Accent), Inter (Body)

## 📋 Estrutura da Página

1. **Hero:** Apresentação do método e proposta de valor.
2. **Pilares (RELACIONAR):** Detalhamento dos 10 pilares estratégicos.
3. **Vídeo:** Seção de conteúdo explicativo.
4. **O Método:** Estrutura do que o cliente recebe (Aulas, Framework, etc).
5. **Níveis de Maturidade:** Sistema de pontuação e classificação das empresas.
6. **FAQ:** Dúvidas frequentes sobre o diagnóstico.
7. **Footer:** Canais de contato e links rápidos.

## 🏁 Como Rodar

1. Instale as dependências:
   ```bash
   npm install
   ```

2. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

3. Abra [http://localhost:3000](http://localhost:3000) no seu navegador.
